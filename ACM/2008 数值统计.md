### 杭电 2008 数值统计



#### code：

```c
#include <stdio.h>
int main(){
	double n,m;
	int a, b, c;
	while (scanf("%lf", &n) != EOF && n != 0) {
		a = b = c = 0;
		for (int i = 0; i < n; i++) {
			scanf("%lf", &m);
			if (m == 0)
				b++;
			else{
				m > 0 ? c++ : a++;
			}
		}
		printf("%d %d %d\n", a, b, c);
	}
}
```
