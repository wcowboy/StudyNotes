### 杭电 2009 求数列的和



#### code:

```c
#include <stdio.h>
#include<math.h>
int main(){
	int n, m;
	double sum,temp;
	while (scanf("%d%d",&n,&m) != EOF){
		sum = 0;
		temp = (double)n;
		for (int i = 0; i < m; i++) {
			sum += temp;
			temp = sqrt((double)temp);
		}
		printf("%.2f\n", sum);
	}
}
```
