`LeetCode` No.206 反转链表.md

### 题目:

> 反转一个单链表。 

### 示例: 

>输入: 1->2->3->4->5->NULL
>输出: 5->4->3->2->1->NULL 

### Related Topics 

>链表 

### 分析

经典的链表反转题，方法有很多，分递归和非递归

#### 非递归

##### 1. 直接遍历

- 创建一个空节点，作为新的头节点，遍历节点
  - 储存当前节点的next值，当前节点next指向新头节点，头节点移动到当前节点，当前节点移动到next下一个节点
- 遍历完成，返回新的头节点

![206-1](D:%5CGithub%5CStudyNotes%5CACM%5CLeetCode%5C206-1.png)

```java
/**
 * @author wcowboy
 * @date 2020/7/9
 * @time 21:43
 */
public class ReverseList {
    public static void main(String[] args) {
        //构造链表
        ListNode l1 = new ListNode(1);
        l1.next = new ListNode(2);
        l1.next.next = new ListNode(3);
        l1.next.next.next = new ListNode(4);
        l1.next.next.next.next = new ListNode(5);
        l1.next.next.next.next.next = new ListNode(6);
        l1.next.next.next.next.next.next = null;

		//调用
        Solution solution = new Solution();
        ListNode l2 = solution.reverseList(l1);

        //输出
        while (l2 != null){
            System.out.println(l2.val);
            l2 = l2.next;
        }
    }

}

class ListNode {
    int val;
    ListNode next;
    ListNode(int x) { val = x; }
}
class Solution {
    public ListNode reverseList(ListNode head) {
        //判空
        if (head == null || head.next == null)
            return head;
        //新建新的头节点listNode
        ListNode listNode = null;
        ListNode cur = head;
        //遍历
        while (cur != null){
            //temp储存cur.next,因为下面的操作会使得cur.next指向别处，从而导致cur无法遍历完毕
            ListNode temp = cur.next;
            cur.next = listNode;
            listNode = cur;
            cur = temp;
        }

        return listNode;
    }
}
```

##### 2. 堆栈

堆栈很好理解，遍历一遍所有数据入栈，然后遍历一遍出栈

```java
import java.util.Stack;

/**
 * @author wcowboy
 * @date 2020/7/9
 * @time 21:43
 */
public class ReverseList {
    public static void main(String[] args) {
        ListNode l1 = new ListNode(1);
        l1.next = new ListNode(2);
        l1.next.next = new ListNode(3);
        l1.next.next.next = new ListNode(4);
        l1.next.next.next.next = new ListNode(5);
        l1.next.next.next.next.next = new ListNode(6);
        l1.next.next.next.next.next.next = null;


        Solution solution = new Solution();
        ListNode l2 = solution.reverseList(l1);


        while (l2 != null){
            System.out.println(l2.val);
            l2 = l2.next;
        }
    }

}


class ListNode {
    int val;
    ListNode next;
    ListNode(int x) { val = x; }
}

class Solution {
    public ListNode reverseList(ListNode head) {
        if (head == null || head.next == null)
            return head;
		//建立堆栈
        Stack<Integer> st = new Stack<Integer>();
        ListNode listNode = head;
        //数据入栈
        while (listNode != null){
            st.push(listNode.val);
            listNode = listNode.next;
        }
        
        //数据出栈
        listNode = head;
        while (!st.isEmpty()){
            listNode.val = st.pop();
            listNode = listNode.next;
        }

        return head;
    }
}
```



#### 递归

递归不同于前面的遍历，递归的交换从最后一个结点开始

![206-2](https://i.loli.net/2020/07/09/wajG4HJmlZL5RT2.png)

```java
/**
 * @author wcowboy
 * @date 2020/7/9
 * @time 21:43
 */
public class ReverseList {
    public static void main(String[] args) {
        ListNode l1 = new ListNode(1);
        l1.next = new ListNode(2);
        l1.next.next = new ListNode(3);
        l1.next.next.next = new ListNode(4);
        l1.next.next.next.next = new ListNode(5);
        l1.next.next.next.next.next = new ListNode(6);
        l1.next.next.next.next.next.next = null;


        Solution solution = new Solution();
        ListNode l2 = solution.reverseList(l1);


        while (l2 != null){
            System.out.println(l2.val);
            l2 = l2.next;
        }
    }

}


class ListNode {
    int val;
    ListNode next;
    ListNode(int x) { val = x; }
}

class Solution {
    public ListNode reverseList(ListNode head) {
        if (head == null || head.next == null)
            return head;

        //listNode为最后一个节点，注意，递归过程中指向不变，每一次返回listNode
        ListNode listNode = reverseList(head.next);
        head.next.next = head;
        head.next = null;

        return listNode;
    }
}
```
